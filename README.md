# Santander Bootcamp 2026 - AI React Front-end

Repositório criado para armazenar projetos, desafios, anotações e exercícios desenvolvidos durante o **Santander Bootcamp 2026 - AI React Front-end**, promovido pela DIO em parceria com o Santander.

---

# 🚀 Sobre o Bootcamp

O bootcamp tem como objetivo ensinar o desenvolvimento de interfaces modernas para aplicações web utilizando tecnologias atuais do mercado, unindo Front-end com Inteligência Artificial.

Durante a trilha, são abordados conceitos fundamentais e avançados de React, TypeScript, UI/UX Design e integração com IA.
